# TechStore Consultores - Plataforma E-commerce Interactiva

Este proyecto es una simulación frontend de una plataforma de comercio electrónico. Su objetivo principal es demostrar la implementación de arquitecturas interactivas utilizando **JavaScript puro (Vanilla JS)**, enfocándose en la manipulación avanzada del Modelo de Objetos del Documento (DOM), la Programación Orientada a Objetos (POO) y la gestión de eventos asíncronos.

## Características Principales (Features)

* **Motor de Búsqueda Dinámico:** Filtrado de productos en tiempo real mediante la captura del evento `input`, actualizando el DOM sin recarga de página.
* **Arquitectura de Datos Orientada a Objetos:** Catálogo estructurado mediante clases de ES6, encapsulando propiedades (precio, stock, categoría) y métodos lógicos de cada producto.
* **Carrito de Compras Reactivo:** Sistema de gestión de estado que calcula totales y descuenta unidades del inventario lógico instantáneamente.
* **Notificaciones Flotantes (Toasts):** Alertas visuales asíncronas y temporizadas para retroalimentar las acciones del usuario (agregados al carrito, falta de stock, promociones aleatorias).

## Stack Tecnológico

* **HTML5:** Estructura semántica del documento.
* **CSS3:** Diseño responsivo, animaciones CSS (`@keyframes`) y variables de estilo.
* **JavaScript (ES6+):** Lógica de negocio, interactividad y manipulación del DOM sin uso de librerías externas.

## Instalación y Ejecución

Al ser un proyecto desarrollado estrictamente en Vanilla JS, no requiere instalación de dependencias ni configuraciones de servidor complejas (Node.js/NPM).

1. Clona este repositorio en tu máquina local:
   ```bash
   git clone [https://github.com/Sergio25-prog/web2_s3.git](https://github.com/Sergio25-prog/web2_s3.git)

1. Abre la carpeta del proyecto en tu editor de código (se recomienda Visual Studio Code).

2. Ejecuta el archivo index.html en tu navegador web preferido, o utiliza la extensión Live Server para una experiencia de desarrollo en vivo.
